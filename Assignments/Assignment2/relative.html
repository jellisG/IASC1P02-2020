  • A link to an external spread sheet works, otherwise it wouldn't look beautiful.
  • Colors: Hyperlinks, "Gateron Yellow" and "BOX Jades"
  • Size: [Holy Pandas]
  • Font: [These models were used with some early 1980s home computers]
  • Capitalize: ["loud and clicky", "loud", and "smooth"]
  • Each of the lists at the bottom are each: Absolute, Relative and Fixed positions.
  • The header and pictures class both require the same change, thus a nested selector.
  • The picture class refers to the img tags, which will only change the 2 images inside that class, unaffecting any other pictures if more are added outside of the class, and thus is a grouped selector.
  • Any div with the wrapper class implemented (including this .html) has a "Box" model. The CSS selector shows margins, border, padding, and the content within the .html.
