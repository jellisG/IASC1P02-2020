Basic Calculator Notes

Practiced the same procedures in a previous lab exercise. The buttons obviously have styles which are similar to the rest of the other buttons throughout the website. Making the traditional calculator would be difficult (and time management) as you would constantly need to consider scenarios of pressing the same operator, "C" function, Square Root, Pi, et cetera. Plagiarism is no good either, so I made my own quick version using function calls. It is a single function which goes through a switch statement rather than going through different function calls. The parameters for the function call is the operator which makes determining which arithmetic operation to do a lot easier. I would have liked to make a modern calculator without user keyboard input, but unfortunately that's for another day.
