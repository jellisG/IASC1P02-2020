This page displays messages with a chosen username, the day and time it was sent, and the message the user put in. These messages came from the HTML page containing inputs and text fields. It saves the data into local storage and every time you load the page or direct yourself to that page, it effectively loads all stored content onto the page for your viewing. Have you noticed that if you reload the page that it doesn’t disappear? Even if you close the page and open it again 5 minutes later, it should still be there. There is also the remove button which removes the message details from local storage and reloads the page to display all the information. Another way to delete everything is to go into Local Storage in the Developer Console and delete the Local Storage value called ‘saved’ and it will delete everything, turning this entire page into an awkward wasteland.
